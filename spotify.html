Overview

This redesign explores a simple idea: users don’t want more recommendations—they want better direction. The UI introduces intent-driven discovery, a spatial layout that feels “navigable”, and high-contrast content blocks that reduce scan fatigue.

Process

I framed the experience around three friction points that show up in everyday listening: decision paralysis, low-signal grids, and personalization that lacks context. The solution focuses on intent controls, curated explanations, and a cinematic hierarchy that guides attention without noise.

Discovery fatigue

Endless choice loops that delay the moment a user actually hits play.

Library clutter

Static structures that scale poorly as collections grow over time.

Lack of intent

Personalization that ignores mood, context, and “why now?” signals.

Solution

The UI system uses soft glass panels, constrained typography, and a Spotify-green highlight to create a “controlled energy” feel. Key moments are designed around: a searchable intent bar, a discovery wheel concept, and insight cards that explain recommendations.
